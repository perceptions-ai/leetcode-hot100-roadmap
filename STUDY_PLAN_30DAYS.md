# C++ Hot100 30天刷题计划（从0到面试）

## 使用规则

1. 每天 90-120 分钟。
2. 每题固定流程：读题 5 分钟，写代码 30 分钟，写测试 15 分钟，复盘 10 分钟。
3. 每题至少 3 组测试：正常样例、边界样例、易错样例。
4. 每周 1 次口述模拟：不运行代码，只讲思路和手推。

## 4周总计划

### 第1周：C++基础 + 哈希/数组入门

1. Day1：两数之和（哈希）
2. Day2：存在重复元素、快乐数（集合/哈希）
3. Day3：有效的字母异位词（字符串 + 计数）
4. Day4：移动零、移除元素（双指针）
5. Day5：区域和检索、和为K的子数组（前缀和）
6. Day6：本周错题重做（不看旧代码重写）
7. Day7：口述模拟（两数之和 + 移动零）

目标：能独立手写哈希 O(n) 解法，并解释复杂度。

### 第2周：链表 + 栈 + 基础二叉树

1. Day8：反转链表
2. Day9：环形链表、链表中点（快慢指针）
3. Day10：有效括号、最小栈
4. Day11：每日温度（单调栈）
5. Day12：二叉树前中后序、最大深度
6. Day13：二叉树层序遍历（BFS）
7. Day14：口述模拟（链表 + 栈）

目标：掌握指针边界和递归终止条件。

### 第3周：二分 + 回溯 + 图搜索

1. Day15：搜索插入位置（二分模板）
2. Day16：查找首末位置、旋转数组搜索（边界二分）
3. Day17：子集（回溯模板）
4. Day18：组合总和（回溯剪枝）
5. Day19：岛屿数量（DFS/BFS）
6. Day20：课程表（拓扑排序）
7. Day21：口述模拟（回溯 + 二分）

目标：形成二分、回溯、BFS/DFS 三个可复用模板。

### 第4周：动态规划 + 面试整合

1. Day22：爬楼梯、打家劫舍（线性DP）
2. Day23：零钱兑换（背包入门）
3. Day24：最长公共子序列（二维DP）
4. Day25：最大子数组和、乘积最大子数组
5. Day26：混合复习（哈希 + 树 + DP）
6. Day27：模拟面试一（45分钟限时）
7. Day28：模拟面试二（不运行，仅手推）
8. Day29：重做全月最难 3 题
9. Day30：整理个人模板库

目标：能完整表达“思路-正确性-复杂度-测试验证”。

## 第1周每日打卡版（详细）

### Day1 两数之和

1. 手写 `vector`、`unordered_map`、函数返回。
2. 完成题目：1. 两数之和。
3. 必测用例：
- `[2,7,11,15], 9 -> [0,1]`
- `[3,2,4], 6 -> [1,2]`
- `[3,3], 6 -> [0,1]`
4. 复盘问题：为什么哈希能把复杂度从 O(n^2) 降到 O(n)。

### Day2 哈希与集合

1. 完成题目：217. 存在重复元素，202. 快乐数。
2. 重点：`unordered_set` 去重，循环检测。
3. 必测用例：
- 空数组
- 单元素
- 全重复元素
4. 复盘问题：什么时候用 set，什么时候用 map。

### Day3 字符串计数

1. 完成题目：242. 有效的字母异位词。
2. 重点：数组计数 vs 哈希计数。
3. 必测用例：
- 正常异位词
- 长度不同
- 含重复字符
4. 复盘问题：固定字符集为什么更适合数组。

### Day4 双指针

1. 完成题目：283. 移动零，27. 移除元素。
2. 重点：快慢指针维护“已处理区间”。
3. 必测用例：
- 全为0
- 不含0
- 0与非0交替
4. 复盘问题：原地算法如何保证不丢数据。

### Day5 前缀和

1. 完成题目：303. 区域和检索，560. 和为K的子数组。
2. 重点：前缀和定义，`sum[i]` 的含义。
3. 必测用例：
- 包含负数
- 全正数
- 目标值为0
4. 复盘问题：为什么前缀和常配合哈希表使用。

### Day6 错题重做

1. 从 Day1-Day5 里选 2 题重写。
2. 不看旧代码，20 分钟限时。
3. 重写后对比旧版，记录差异。

### Day7 口述模拟

1. 模拟面试 30 分钟。
2. 口述两题：两数之和 + 移动零。
3. 固定话术：思路、复杂度、边界测试、可能 bug 点。

## 面试训练模板

1. 先给暴力思路，再给优化思路。
2. 明确数据结构选择原因。
3. 主动给 3 组测试并手推变量变化。
4. 最后总结时间复杂度和空间复杂度。
